Further reading

A short, curated set of links for going deeper: the City's live air-quality map for current conditions, the health case for caring about traffic-related air pollution, the open dataset this site is built on, and the source code for the site itself. More may be added over time.

Live air quality

  • Montréal real-time air-quality map — The City of Montréal's official map of current air-quality conditions across the RSQA network. Where this site shows the historical record, that map shows what the stations are reading right now — the latest hourly readings and Air Quality Index (IQA) at each station. Bilingual (FR/EN).

Health

  • Health impacts of traffic-related air pollution in Canada — Health Canada's assessment of how pollution from road traffic affects health. It estimates that roughly four in ten Canadians live within 250 m of a high-traffic road and attributes about 1,200 premature deaths a year to traffic-related air pollution, the mixture of fine particulates and gases (including NO and NO₂) that this site maps. The linked section discusses near-road exposure specifically.

Data

  • RSQA — gaseous pollutants (Données Montréal) — The City of Montréal's open-data publication of the RSQA network's continuous hourly measurements of gaseous pollutants (NO, NO₂, O₃, SO₂, CO and others). These are the raw measurements this visualizer is built on; the dataset page links to the source CSV archives and the data licence.

Code

  • OpenMobilityData / AirQualityMTL on GitHub — The complete source code for this site: the Rust/WebAssembly front end and the scripts that fetch and preprocess the RSQA archive. It is public so the methodology can be scrutinized and others can learn from how it was built; it is not, however, run as a collaborative project, so issues and pull requests may go unanswered.
